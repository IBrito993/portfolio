# **App Name**: Brito Portfolio

## Core Features:

- Home/Hero Section: Display a welcome message, introduction, and links to key sections of the portfolio.
- About/Bio Section: Showcase Ivan Brito's background, skills, and technologies. Display skills using icons.
- GitHub Projects: Fetch and display Ivan Brito's top repositories from GitHub (https://github.com/IBrito993) :contentReference[oaicite:2]{index=2}, including project name, description, tech stack, and links.
- Resume/Experience Timeline: Present a professional timeline of Ivan Brito's work history, education, and skills breakdown.
- Contact/Connect Links: Provide links to Ivan Brito's GitHub (https://github.com/IBrito993) and LinkedIn (https://www.linkedin.com/in/ibrito93/) :contentReference[oaicite:3]{index=3}. Offer an optional contact form.
- Theme Toggle: Allow users to switch between a light and dark theme for the portfolio site, storing theme preference in local storage.
- SEO Tool: Use an LLM tool to produce high quality SEO text that reflects current best practices and relevant keywords for each section of the website.

## Style Guidelines:

- Primary color: Dark desaturated blue (#3B495B) to provide a professional, technological, yet somewhat somber, impression, suitable to software engineering.
- Background color: Very dark desaturated blue (#1A1F26) to reinforce the dark theme.
- Accent color: A more saturated blue (#4385DE) for interactive elements and highlights, analogous to the primary, but much brighter.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look. It will be suitable for both headlines and body text.
- Use a set of consistent and modern icons for skills, technologies, and social media links.
- Maintain a clean and minimal layout with clear sections, consistent spacing, and responsive design for different devices.
- Incorporate subtle animations on hover or scroll to provide a smooth and engaging user experience.
